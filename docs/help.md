### Keyboard Shortcuts / Hotkeys

* Help: <kbd>Shift</kbd> + <kbd>?</kbd>

* Hard Refresh: <kbd>Ctrl</kbd> + <kbd>R</kbd>

> ⚠️**CAUTION**⚠️
>
> If the focus is not on the document then the browser will apply its default action for the shortcut!

* Close/Escape: <kbd>Ctrl</kbd> + <kbd>X</kbd>

* Search: <kbd>Ctrl</kbd> + <kbd>K</kbd>

* Configuration: <kbd>Ctrl</kbd> + <kbd>,</kbd>

&nbsp;

* Console: <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>C</kbd>

* Cache: <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>D</kbd>
